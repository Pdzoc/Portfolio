Hello!

My name is Piotr and I'm a code hobbyist from Poland.

I learn programming (front-end web development) since 2019, although I created my first site in 2007 (don't set your expectations too high - I was a teenager). It was "Bractwo Żelaza" (English: Brotherhood of Iron) - dedicated to the RPG Fallout series. I don't have these files, but you can still see it (unfortunately without images) on a Wayback Machine.

I prefer useful things, hence most of my projects include a "teaching" element. Mostly basic math and English, because such projects were created for my nephews, but sometimes I wrote tools for myself and other users (e.g. Word List).

My (exactly: my sister's) first computer had Windows 98, so I decided to relate to this style.
